# TBCC

A C compiler following the [book](https://norasandler.com/book/) by Nora Sandler.

### Progress

- [x] Chapter 1: Constant integers
    - [x] Lexing a basic source file
    - [x] Parsing the produced tokens
    - [x] Converting the C abstract syntax tree into an assembly abstract syntax tree.
    - [x] Generating assembly code from the tree.
    - [x] Running tests provided by Nora Sandler
- [ ] Chapter 2: Unary Operators
  - [ ] Lexing tokens for negation and bitwise complement.
  - [ ] Parsing negation and bitwise complement.
  - [ ] Conversion of program into TACKY intermediate representation.
  - [ ] Conversion of TACKY into assembly abstract syntax tree.
  - [ ] Generating assembly for new unary operators.
  - [ ] All Nora Sandler tests pass.
